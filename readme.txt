SnapCode - Run WordPress code instantly without a code editor

== Description ==

SnapCode empowers you to execute PHP code directly within your WordPress project without the need for an external editor.

Features

* Easy and simple UI
* Execute PHP and WordPress code instantly
* Test any plugin code snippet or helper
* Get immediate output
* Shows queries for executed code.
* Skip the step of creating a separate PHP file for testing function calls
